# Codex No Micro Launcher v0.1.0

Windows용 Microsoft Store Codex의 선택적 Codex Micro/Work Louder HID 탐색으로 인한
주기적 UI 정지를 우회하는 첫 공개 버전입니다.

## 주요 기능

- 원본 Store Codex를 수정하지 않는 버전별 복제 런타임
- 확인된 장치 탐색 호출 한 곳만 동일 바이트 길이로 비활성화
- 패턴 수, 파일 길이, SHA-256 검증
- Store 업데이트 감지와 새 버전 런타임 자동 준비
- 바탕화면 바로가기와 제거 기능

## 설치

1. `Codex-No-Micro-v0.1.0.zip`을 내려받아 압축을 풉니다.
2. 일반 Codex를 트레이까지 완전히 종료합니다.
3. `Install.cmd`를 실행합니다.
4. 바탕화면의 **Codex (No Micro)**를 사용합니다.

## 호환성

- 확인된 Codex Store 버전: `26.715.4045.0`
- Windows 10/11 x64용
- 비공식 커뮤니티 도구이며 OpenAI와 관련이 없습니다.

새 Codex 버전에서 내부 패턴이 바뀌면 런처는 수정하지 않고 안전하게 중단합니다.
